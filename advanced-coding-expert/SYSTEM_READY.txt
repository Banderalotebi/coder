Issue: "Can't import phase2_rag module"
Fix: Add to .bashrc or .zshrc:
     export PYTHONPATH=/Users/bander/coder/advanced-coding-expert:$PYTHONPATH

Issue: "Ollama connection refused"
Fix: Check if running: curl http://localhost:11434/api/tags
     Or restart: killall ollama && ollama serve

Issue: "API server won't start"
Fix: Check port 8000 is free: lsof -i :8000
     Use different port: python -m uvicorn api_server:app --port 8001

Issue: "Model loading is slow"
Fix: This is normal first time (loading 4.9GB into memory)
     Subsequent loads are cached
     Use 7B models for faster responses
